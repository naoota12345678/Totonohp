FEATURES
FEATURE 01
EC売上の自動取込・集計
楽天・Amazon・カラーミーをAPI連携。エアレジ等CSVエクスポートにも対応し、EC＋BtoB売上を日別・月別・事業部別に自動集計します。
CSV一括取込対応
FEATURE 02
商品マスタ・取引先別単価
商品・取引先・単価マスタをCSVで一括登録。既存データをそのまま移行でき、導入初日からフルで使えます。会計ソフト連携のCSV出力にも対応。
移行コストゼロ
FEATURE 03
定期請求の完全自動化
毎月・隔月・四半期の定期課金を自動処理。自動送信/承認後送信/手動の3モードを取引先ごとに設定できます。
FEATURE 04
未入金アラート・延滞管理
1ヶ月未入金で自動アラートを発報。経過日数を色分け表示し、ダッシュボードで延滞状況を一覧把握。回収漏れを確実に防止します。
回収漏れゼロ
FEATURE 05
メール送付・送信履歴管理
請求書をワンクリックでメール送付。メール・郵送・FAX・手渡しの送付方法を記録し、督促・再送もすぐ対応できます。
FEATURE 06
請求書承認ワークフロー
送付前に担当者→管理者の承認フローを設定。承認待ち一覧から差し戻し・承認をワンクリックで処理できます。
FEATURE 07
Stripe決済で未入金を確実回収
未入金の取引先にオンライン決済リンクを送信。代行手数料ゼロ・通常の決済手数料もゼロ、未入金時のみStripe（3.6%）で回収します。
FEATURE 08
事業所ごとの請求設定
事業所・事業部単位で請求元・口座・インボイス番号を個別設定。複数ブランド・複数事業を持つ企業に最適です。
FEATURE 09
消費税率の完全フリー設定
10%・8%（軽減税率）・0%、将来の税率変更にも即対応。品目ごとに税率を自由設定し、混在処理も自動計算します。
食品・飲料業に最適
請求代行 不要
NP掛け払いなどの請求代行は、代行手数料（平均約3%）に加えて決済手数料も常時かかります。本システムなら代行手数料はゼロ。未入金が発生した場合のみStripe決済（3.6%）の決済手数料で確実に回収できます。月商500万円なら年間100万円以上のコスト削減になります。
